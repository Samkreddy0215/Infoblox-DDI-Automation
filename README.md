## Automation Scripts

### Infoblox IPAM Report

This script connects to the Infoblox WAPI REST API to collect IP network information and generate an IPAM utilization report.

### Features

- Connects to Infoblox WAPI
- Retrieves configured IP networks
- Exports results to CSV
- Uses REST API authentication
- Supports enterprise IPAM reporting
- DNS Forwarding and Delegation
